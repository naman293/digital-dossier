# 🎮 Secret Mini-Game Proposals

Here are 5 technical, retro-styled mini-games that would fit perfectly as a hidden Dossier Easter Egg. Each can be built with React and Framer Motion.

## 1. 🐍 Terminal Snake
The classic Snake game, but rendered entirely in the ASCII-style monospace font.
- **Aesthetic**: Deep crimson "body" with a glowing white "food" node.
- **Challenge**: Collect "Data Packets" to increase your "Dossier Byte Size".
- **Twist**: The grid is a terminal buffer; hitting the edge "corrupts" the system.

## 2. ⌨️ Hacker Type-Racer
A fast-paced typing challenge where you must accurately type out C++ or Python code snippets.
- **Aesthetic**: Lines of code scroll vertically like a real IDE.
- **Goal**: Reach high WPM (Words Per Minute) to "Bypass the Firewall".
- **Reward**: Unlocks a "Hacker Elite" badge in the UI.

## 3. 🛡️ Dossier Invaders
A Space Invaders clone where you defend the "Dossier Core" from falling "Security Threats" (bug icons).
- **Aesthetic**: Simple geometric shapes and CRT scanline effects.
- **Mechanic**: Use Arrow keys to move and Space to "patch" the bugs.
- **Difficulty**: Speed increases with every wave cleared.

## 4. 🧩 Circuit Connector
A spatial logic puzzle where you must rotate pipe-like circuit pieces to connect the "Source" to the "Database".
- **Aesthetic**: Glowing neon lines over a dark geometric grid.
- **Goal**: Complete the circuit in the fewest moves possible.
- **Feeling**: Very high-tech, satisfying click sounds on every rotation.

## 5. 🔓 Hash-Cracker (Mastermind)
A deduction game where you must guess the 4-digit secret "System Password".
- **Aesthetic**: Sleek UI with "Encrypted" placeholder characters.
- **Logic**: After each guess, the system tells you how many digits are correct and in the right position.
- **Experience**: Feels like you are actually brute-forcing a secure server.
