# Design System: Digital Dossier

## Aesthetic Philosophy
The design follows a **Neo-Technical Dossier** (or "Cyber-Terminal") aesthetic. It aims to evoke the feeling of a high-security digital file or a sophisticated terminal interface, characterized by:

- **Monospace Dominance**: Using technical fonts for data presentation.
- **Deep Contrast**: Dark backgrounds balanced with vibrant, high-energy accents.
- **Micro-Detailing**: Small decorative elements (corner borders, circuit lines, dots) that suggest complexity and precision.

## Visual Language

### Color Palette (OKLCH)
- **Primary (Crimson)**: `oklch(0.50 0.18 18)` — A sharp, aggressive red for key actions and highlights.
- **Background (Obsidian)**: `oklch(0.13 0.005 260)` — A deep, slightly bluish obsidian for maximum readability.
- **Surfaces**: `oklch(0.16 0.005 260)` — Subtle elevation layers.
- **Grid Lines**: `oklch(0.22 0.005 260)` — Structural guides that provide a "blueprint" feel.

### Typography
- **Display/Technical**: `JetBrains Mono` — Crisp, legible, and distinctly "code."
- **Body**: `Inter` — High-readability sans-serif for long-form content.

### Signature UI Elements
1. **Scanline Overlay**: A persistent `fixed` overlay that adds a retro-CRT texture.
2. **Dossier Panels**: Bordered containers with highlighted "L-bracket" corners.
3. **Status Indicators**: Pulsing dots and flickering animations to make the UI feel "alive" and reactive.
4. **Circuity Lines**: Decorative lines with terminal nodes to connect sections visually.

## Future Advancements (WOW Factor)
To enhance the premium feel, consider these additions:

- **Dithered Transitions**: Use dithered image loading or noise-based page transitions.
- **Audio Feedback**: Suble mechanical "clicks" on button hovers or terminal-style text-typing sounds.
- **Ambient Noise**: A very low-opacity grainy texture or "magnetic interference" effect that reacts to cursor movement.
- **Biometric Animations**: Faint "fingerprint" or "retinal scan" motifs in the Hero or Contact sections.
- **Glitch Effects**: Rare, subtle displacement map filter glitches on hover of specific dossier titles.
